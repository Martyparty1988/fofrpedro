# Runner assets

The bundled runner assets are sourced from Mesh2Motion's CC0 collection:

- `runner-human-sophia.glb` — primary detailed character,
- `runner-human-base.glb` — lightweight fallback,
- `runner-animations.glb` — seven selected gameplay animations without root translation.

Their exact source, pinned revision, and license are documented in
[`THIRD_PARTY_ASSETS.md`](../../THIRD_PARTY_ASSETS.md).
